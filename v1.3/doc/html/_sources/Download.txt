Getting the SyncPy library
==========================

Dependencies 
------------

For a complete use of the **SyncPy** library, please install these libraries first :

* `Python`_ (v2.7)
* `Matplotlib`_ (v1.4.3)
* `NetworkX`_ (v1.10)
* `Numpy and Scipy`_ (v1.9) 
* `Pandas`_ (v 0.16.0)
* `Statsmodels and Patsy`_ (v0.6.1)

.. _Python: https://www.python.org/download/releases/2.7/
.. _Matplotlib: http://matplotlib.org/downloads.html
.. _NetworkX: https://networkx.github.io/download.html
.. _Numpy and Scipy: http://www.scipy.org/scipylib/download.html
.. _Pandas: http://pandas.pydata.org/pandas-docs/stable/install.html
.. _Statsmodels and Patsy: http://statsmodels.sourceforge.net/install.html

To correctly use the interface, please download also this library:

* `pyQt`_ (v4.11.3)

.. _pyQt: http://pyqt.sourceforge.net/Docs/PyQt4/installation.html

Version numbers are just an indication to precise which versions have been used to test the current release.
